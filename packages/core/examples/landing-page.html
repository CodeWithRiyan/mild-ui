Build Beautiful UIs with mild-ui

Framework-agnostic CSS utilities and components. Tailwind-like utilities with the mild- prefix.

Why Choose mild-ui?

Built for modern web development with flexibility and performance in mind

Lightning Fast

Pure CSS with zero JavaScript. Minimal bundle size and optimal performance.

Framework Agnostic

Works with React, Vue, Angular, or vanilla HTML. Use anywhere.

Highly Customizable

SCSS variables, design tokens, and CSS custom properties for full control.

Easy to Use

Just add the CSS file and start using utility classes

1. Install & Import

# Install via npm
npm install @mild-ui/core

# Import in your CSS
@import '@mild-ui/core/dist/mild-ui.css';

# Or use CDN
<link rel="stylesheet" href="path/to/mild-ui.css">

2. Use Utility Classes

<div class="mild-flex mild-items-center mild-gap-4">
  <img class="mild-w-12 mild-h-12 mild-radius-full" 
       src="avatar.jpg" alt="Avatar">
  <div>
    <h3 class="mild-font-semibold mild-text-gray-900">
      John Doe
    </h3>
    <p class="mild-text-gray-600 mild-text-sm">
      Web Developer
    </p>
  </div>
</div>
