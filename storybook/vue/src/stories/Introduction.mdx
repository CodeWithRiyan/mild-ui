# Welcome to mild-ui Vue

mild-ui is a cross-framework UI component library inspired by the copy-paste approach of shadcn/ui, extended to work across React and Vue.

## Getting Started

Components are designed to be copied into your project rather than imported as dependencies. Use our CLI tool to add components:

```bash
# Install the CLI globally
npm install -g @mild-ui/cli

# Add components to your project
mild-ui add button --framework=vue
mild-ui add input --framework=vue
```

## Features

- **Cross-framework compatibility**: Works with React and Vue
- **Framework-agnostic core**: Consistent design and behavior
- **Tailwind CSS**: Modern utility-first styling
- **Accessibility**: Built with accessibility in mind
- **TypeScript**: Full TypeScript support
- **Vue 3 Composition API**: Modern Vue development patterns
- **Customizable**: Easy theming and customization

## Vue-Specific Features

- **Composition API support**: Built with Vue 3 Composition API
- **Script setup syntax**: Modern single-file component structure
- **v-model integration**: Proper two-way data binding
- **Slot-based architecture**: Flexible content projection
- **Reactive props**: Full reactivity with Vue's reactive system

## Available Components

- **Button**: Primary, secondary, outline, and ghost variants with icon support
- **Input**: Text input with validation states and v-model support
- **Checkbox**: Checkbox with indeterminate state support
- **Radio**: Radio buttons and radio groups
- **Select**: Dropdown select with search and multi-select capabilities

```vue
      <template #leading-icon>

    <Checkbox v-model="isChecked" label="Accept terms" />
  </div>
</template>

<script setup lang="ts">
import { ref } from "vue";
import { Button, Input, Checkbox } from "./components";
import { PlusIcon } from "lucide-vue-next";

const inputValue = ref("");
const isChecked = ref(false);
const hasError = ref(false);

const handleClick = () => {
  console.log("Button clicked!");
};
</script>
```
