## Getting Started

Install components using the mild-ui CLI:

````bash
# Install a single component
mild-ui add Button --framework=react

# Install all components
mild-ui add --all --framework=react

## Theming

All components support theming through CSS variables:

```typescript
css:root {
  --primary: 222.2 47.4% 11.2%;
  --primary-foreground: 210 40% 98%;
  /* More theme variables... */
}
````

Use the ThemeProvider to enable theme switching:

```typescript
tsximport { ThemeProvider } from '@mild-ui/react';

function App() {
  return (
    <ThemeProvider defaultTheme="system">
      <YourApp />
    </ThemeProvider>
  );
}
```
